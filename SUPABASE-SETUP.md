# SnowBall Merchants - Supabase Setup Guide

## Step 1: Create Supabase Project
1. Go to https://supabase.com and sign in
2. Click "New Project"
3. Fill in details:
   - Name: SnowBall-Merchants
   - Database Password: (create a strong password)
   - Region: India (or closest to you)
4. Wait for project to initialize (2-3 minutes)

## Step 2: Get API Credentials
1. Go to Project Settings → API
2. Copy:
   - Project URL (e.g., https://xxxxx.supabase.co)
   - anon public key

## Step 3: Configure SMS (for OTP)
1. Go to Authentication → Providers → Phone
2. Enable "Phone Auth"
3. Configure SMS provider (choose one):
   - **Twilio** (recommended - free trial available)
   - **Msg91** (popular in India)
   
### Twilio Setup:
1. Create free account at https://twilio.com
2. Get Account SID, Auth Token, and verify phone number
3. Go to Supabase → Authentication → Providers → Phone
4. Enter Twilio credentials

## Step 4: Create Enquiries Table
1. Go to Supabase → SQL Editor
2. Run this SQL:

```sql
-- Create enquiries table
CREATE TABLE IF NOT EXISTS enquiries (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    contact_person TEXT NOT NULL,
    firm_name TEXT,
    mobile TEXT NOT NULL,
    email TEXT NOT NULL,
    requirements TEXT NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Enable RLS
ALTER TABLE enquiries ENABLE ROW LEVEL SECURITY;

-- Allow public insert
CREATE POLICY "Allow public insert" ON enquiries
    FOR INSERT TO anon, authenticated
    WITH CHECK (true);

-- Allow public read (for admin)
CREATE POLICY "Allow public read" ON enquiries
    SELECT TO anon, authenticated
    USING (true);
```

## Step 5: Update Website
1. Open index.html
2. Find and replace these lines (around line 2317-2319):
```javascript
const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_ANON_KEY = 'your-anon-key';
```
With your actual credentials:
```javascript
const SUPABASE_URL = 'https://xxxxx.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
```

## Step 6: Deploy
1. Push changes to GitHub
2. GitHub Pages will auto-deploy

## Testing
1. Open your website
2. Fill enquiry form
3. Click "Send OTP" - you should receive SMS
4. Enter OTP and verify
5. Submit enquiry - should be saved in Supabase

## Checking Enquiries
1. Go to Supabase → Table Editor → enquiries
2. You can see all submitted enquiries
